# Sales & Inventory Management Analysis 📊

### Project Overview
This project focuses on managing inventory levels and analyzing sales performance for Ganga-Mart. It provides a clear overview of product availability and pricing, helping to optimize inventory management and sales strategies.

### Key Features
*   **Inventory Tracking:** Monitors stock levels to distinguish between "Available" and "Out of Stock" items.
*   **Price Analysis:** Includes calculated metrics such as Minimum, Maximum, and Average product prices.
*   **Data Visualization:** Features interactive charts (Pie Charts and Bar Graphs) to visualize stock distribution and pricing trends.

### Tools & Technologies Used
*   **Microsoft Excel:** Used for data entry, Pivot Tables, and creating visualizations.

### Key Insights
*   **Stock Distribution:** 63.6% of items are currently "Available," while 36.4% are "Out of Stock," indicating a need for replenishment.
*   **Pricing Trends:** Detailed analysis of various price points to understand product value distribution.

### Future Improvements
*   Implement automated low-stock alerts.
*   Add a date-tracking column to analyze monthly sales performance.
